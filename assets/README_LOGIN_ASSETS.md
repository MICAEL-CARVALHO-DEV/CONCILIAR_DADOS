# Assets do Login (SaaS)

Use estes nomes de arquivo para o login funcionar sem alterar código:

- `assets/login-bg.jpg` (fundo 16:9, ideal 1920x1080 ou maior)
- `assets/sec-logo.png` (logo/brasão com fundo transparente)

## Prompt recomendado (fundo institucional)

```text
modern educational campus building with palm trees at sunset, clean contemporary architecture, wide entrance pathway, soft warm sunlight, tropical environment similar to Brazil, modern glass and concrete school building, cinematic lighting, depth of field, slightly blurred background for login page, professional institutional atmosphere, high quality, 16:9, 1920x1080, soft background blur, shallow depth of field, low visual noise, ideal for website login background
```

## Prompt alternativo (mais clean para UI)

```text
clean modern university campus with palm trees and minimal landscaping, large glass building and concrete structure, wide pathway leading to entrance, tropical environment at golden hour sunset, soft warm lighting, shallow depth of field blur to support UI overlay, minimal visual noise, ideal background for SaaS login page, ultra realistic, 16:9, 1920x1080
```

## Configuração de geração

- Aspect ratio: `16:9`
- Resolução: `1920x1080` ou `4K`
- Estilo: `realistic`
- Luz: `golden hour`

## Passos para concluir visual SaaS

1. Gere 3 a 5 variações do fundo com os prompts.
2. Escolha a mais limpa para leitura do formulário.
3. Salve como `assets/login-bg.jpg`.
4. Exporte o brasão em PNG transparente e salve como `assets/sec-logo.png`.
5. Recarregue o browser com `Ctrl+F5`.

Prompts adicionais: `assets/prompts_login_saas.md`

## Critério de aceite visual

- Texto legível em desktop e mobile.
- Formulário com contraste alto.
- Fundo bonito sem poluição visual.
- Login mantém fluxo atual (sem quebrar autenticação).
