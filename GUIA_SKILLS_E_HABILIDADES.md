# GUIA SKILLS E HABILIDADES (PROJETO + IA)

Data de atualizacao: 2026-03-04

## 1. Skills disponiveis no seu ambiente Codex
No AGENTS.md desta sessao, as skills ativas sao:

1. `skill-creator`
- Uso: criar/atualizar uma skill especializada.
- Quando chamar: quando voce quiser empacotar um metodo recorrente (ex.: playbook de deploy, playbook de homologacao).

2. `skill-installer`
- Uso: instalar skills no seu ambiente Codex.
- Quando chamar: quando voce quiser trazer uma skill pronta de lista curada ou repositorio.

Resumo pratico:
- Hoje voce ja "tem" essas 2 skills de sistema disponiveis.
- Elas sao para evoluir seu proprio ambiente de automacao, nao para substituir seu estudo tecnico.

## 2. Habilidades tecnicas que voce ja demonstrou
- Visao de produto (prioridade por valor real)
- Evolucao de frontend (UX de login/cadastro)
- Deploy e operacao (Render + front estatico)
- Investigacao de erro com logs
- Persistencia para fechar fluxo ponta a ponta

## 3. Habilidades que faltam fortalecer (para full stack forte)
- Debug orientado a causa raiz (nao sintoma)
- Modelagem de dados e migracoes sem quebrar ambiente
- Seguranca aplicada (CORS, token, segredo, LGPD)
- Teste funcional repetivel (checklist e evidencias)
- Comunicacao executiva para chefia

## 4. Plano de evolucao em 2 modulos de estudo
### Modulo 1 - Projeto CONCILIAR_DADOS
Meta:
- explicar arquitetura, riscos e fluxo operacional para chefia em linguagem simples.

Entregaveis:

- checklist de homologacao
- roteiro de demonstracao

### Modulo 2 - Programar com IA
Meta:
- usar IA para acelerar entrega sem perder dominio tecnico.

Entregaveis:
- prompt padrao de implementacao
- prompt padrao de review
- padrao de evidencia e validacao

## 5. Regra de maturidade tecnica
- IA te ajuda a codar mais rapido.
- Voce responde por seguranca, arquitetura, teste e decisao final.

## 6. Como evoluir sem se perder
- Toda tarefa com criterio de aceite
- Toda mudanca com evidencia
- Todo erro com causa raiz registrada
- Toda semana com revisao de aprendizado
